# Readme

This independent study project is intended to align an intensive review of Linear Algebra, Statistics,
and Trigonometry with practice learning the Python modules Matplotlib, Numpy, and (to an extent) SciPy.
These are notoriously more difficult as libraries to work with compared to the libraries built on top of
them (e.g. seaborn, state-based pyplot, and pandas). These subjects offer great opportunities to train
both the foundational math skills needed for data analysis, but also allow for a ground-up study of the
engines of data analysis and visualization in Python, NumPy and Matplotlib. This repository is intended
to contain periodic artifacts, larger scale projects, that allow for demonstration of the learning gathered
from these studies, through notebooks that work through complex, difficult math problems and projects using
These Python tools.
